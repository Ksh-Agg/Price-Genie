# PriceGenie

PriceGenie is a machine learning-based price comparison and recommendation system that analyzes product datasets and predicts optimal pricing using CatBoost.

## Features

* Product price analysis using machine learning
* CatBoost model for prediction
* Large product dataset across multiple categories
* Jupyter Notebook implementation

## Tech Stack

* Python
* Jupyter Notebook
* CatBoost
* Pandas
* Scikit-learn

## Project Structure

```
Dataset/        -> Product datasets
catboost_info/  -> CatBoost training logs
Data.ipynb      -> Main training and analysis notebook
```

## How to Run

1. Clone the repository

```
git clone https://github.com/YOUR_USERNAME/Price-Genie.git
```

2. Install dependencies

```
pip install pandas scikit-learn catboost
```

3. Open the notebook

```
jupyter notebook Data.ipynb
```
